Add simple date and time in text form using the shortcode.

== Description ==
A plugin from which you can easily display current date and time on your WordPress site. Widget, Post, Page Header, Footer, Anywhere. Just place the shortcode and you can choose any time zone you want. 

It also work with every kind of cache.

== Installation ==
Upload and extract simple-date-and-time-xs.zip to the /wp-content/plugins/directory OR
Upload simple-date-and-time-xs.zip to the Plugins > Add New > Upload
Activate the plugin through the 'Plugins' menu in WordPress
You are good to go,  Go to Plugin About to learn how to use it.
